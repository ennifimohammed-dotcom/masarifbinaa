# 🏗️ ENNIFI - تطبيق متابعة مصاريف البناء

تطبيق Android احترافي لمتابعة نفقات بناء منزلك بالعربية والفرنسية.

---

## 📱 مميزات التطبيق

- ✅ **لوحة تحكم** — إجمالي المصاريف، شريط تقدم الميزانية، رسوم بيانية
- ✅ **قائمة النفقات** — بحث وفلترة حسب الفئة، حذف بالسحب
- ✅ **إضافة نفقة** — مع تاريخ، صنف، ملاحظات، صورة الفاتورة
- ✅ **إحصاء وتقارير** — توزيع الفئات، تصدير PDF
- ✅ **حماية برمز PIN** — 6 أرقام مشفرة
- ✅ **ثنائي اللغة** — العربية والفرنسية بنقرة واحدة
- ✅ **تنبيه الميزانية** — عند الاقتراب من 90% أو التجاوز
- ✅ **حفظ محلي** — SQLite بدون إنترنت

---

## 🛠️ كيفية تثبيت وتشغيل التطبيق

### المتطلبات
- Flutter SDK (3.0+) — https://flutter.dev/docs/get-started/install
- Android Studio أو VS Code
- جهاز Android أو محاكي

### خطوات التثبيت

```bash
# 1. تثبيت Flutter (إن لم يكن مثبتاً)
# اتبع التعليمات على: https://docs.flutter.dev/get-started/install/windows

# 2. فتح المجلد
cd ennifi_construction

# 3. تحميل المكتبات
flutter pub get

# 4. تشغيل التطبيق على جهاز متصل أو محاكي
flutter run

# 5. بناء APK للتثبيت المباشر
flutter build apk --release
# ستجد الملف في: build/app/outputs/flutter-apk/app-release.apk
```
